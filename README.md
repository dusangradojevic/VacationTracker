## Install library

    cd database
    mvn clean install
    
## Run

    docker-compose up 
    
# REST API

The REST API is described below.

## Import Employee Profiles

### Request

`POST /api/upload/importEmployeeProfiles`

    curl -i -H 'Accept: application/json' http://localhost:8081/api/upload/importEmployeeProfiles
